## ECHELON Evolution

The lore documents establish the fictional context for the project's evolution:

1. **Pre-War Development** (2075-2077)
   - Excel prototype (Dr. Krane's research)
   - SPARROW - DOS proof of concept
   - RAVEN - First operational prototype
   - GHOST - SIGNET network deployment
   - ECHELON - Pip-Boy field version

2. **Post-War Legacy** (2077-2287)
   - Survival through the Great War
   - Continued development at Raven Rock
   - Modern usage by Enclave remnants
