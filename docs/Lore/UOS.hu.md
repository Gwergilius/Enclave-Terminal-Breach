# Unified Operating System (Egységes operációs rendszer)

**[English]** | Magyar

> Összefoglaló a [Unified Operating System] cikkből (fallout.fandom.com). Eredeti forrás: [Unified Operating System].

Az __Unified Operating System__ (UOS) szöveges felhasználói felületű operációs rendszer, a [RobCo Industries] fejlesztette; kezeli a számítógép hardverét és szoftverkörnyezetként szolgál. Mainframe-en fut.

## Háttér

Az [UOS][Unified Operating System] a RobCo 2075-ben publikálta, lejárat [2077]. Az UOS az MF Boot Agentre és a RETROS BIOS-ra támaszkodik. A RobCo Industries Termlink protokollt használják bejelentkezéshez és elveszett jelszavak helyreállításához; ez a fő eszköz a [hacking][Minigame] – terminál feltörés – korlátozott tartalom eléréséhez.

## Parancsok

Rendszer fejléc: ROBCO INDUSTRIES UNIFIED OPERATING SYSTEM, COPYRIGHT 2075-2077. Logon: WELCOME TO ROBCO INDUSTRIES (TM) TERMLINK, LOGON ADMIN, ENTER PASSWORD NOW. Hacking: SET TERMINAL/INQUIRE, MF Boot Agent, RETROS BIOS, RUN DEBUG/ACCOUNTS.F. Minijáték: TERMLINK PROTOCOL, ENTER PASSWORD NOW.

## Termlink

Bejelentkezés, elveszett jelszavak helyreállítása, hacking fő eszköze. Hibakódok 0x + nyolc változó; külső eszköz hibánál rövidebb külső hibakódok (0x + három változó).

## Megjelenések

[Fallout 3][FO3], [Fallout: New Vegas][FNV], [Fallout 4][FO4], [Fallout 76][FO76].
