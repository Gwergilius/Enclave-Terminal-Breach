# Terminál paletta színek

![Palette Colors]

## Zöld monitor
Klasszikus monokróm zöld foszfor paletta (Hercules stílus). `GreenPalette`: Background, Dark, Normal, Bright – lásd angol [Palette.md](Palette.md) C# kód.

## Borostyán monitor
Klasszikus monokróm borostyán foszfor paletta. `AmberPalette`: Background, Dark, Normal, Bright.

## Fehér monitor
Klasszikus monokróm fehér foszfor paletta. `WhitePalette`: Background, Dark, Normal, Bright.

## Kék monitor
Klasszikus monokróm kék foszfor paletta. `PhosphorBluePalette`: Background, Dark, Normal, Bright.

Teljes C# kód mindegyik palettához: angol [Palette.md](Palette.md).
